changelog
2024-09-02
Updates
  • Add new block types: moving-rotating and moving-swinging
  • Add option for dynamic objects to be recreated when destroyed
  • Add 2 blocks: Bean with swim ring, inflatable rotor
  • Minor bugfixes
2023-07-07
Updates
  • New blocks - urban theme
2023-06-05
Updates
  • Randomness for competitive/elimination levels: define variables and use them to change block properties when the round is reset
  • More blocks supported for object spawners (e.g. barrels) and added setting for initial rotation of spawned objects
  • Minor bugfixes and UI improvements (e.g. prevent setting scale to zero or too high, show red text when hovering a spawn colliding with decor)
2023-05-15
Updates
  • New level types: competitive race & elimination
2023-04-19
Updates
  • Allow kinematics for blocks without a collider
  • Export/import groups (copy/paste between levels)
  • Multi-select: add to selection (and remove) with SHIFT key
  • Display limits for publishing (top-right corner)
  • Bugfixes & small UI improvements
2023-04-07
Updates
  • New feature: Multi-select to work with multiple blocks at the same time
  • New feature: Group management to save block selections
  • support changing texture for spawned objects (cubes)
  • Bugfixes
2023-03-27
Updates
  • New in level settings : change background skydome
  • New in level settings : change level type and countdown start interval (timed race)
  • Fix respawn in gravity blocks
  • Do not change camera distance when moving/cloning
  • Minor bugfixes
2023-03-17
Updates
  • New feature: Teleport (better than checkpoint-kill 😅)
  • 10 new textures
  • 50 new blocks (decor and obstacles)
  • Option for jumpads to unshock player when it's triggered (avoids getting stuck forever on vertical jumpads)
  • Better sliding controls: now relative to camera instead of bean orientation. Also better sliding initial player bump and make sliding work with altered gravity
  • Added option to change gravity strength
  • various bugfixes you probably don't care about because nobody reported them
2023-03-09
Updates
  • New option for jumpads to affect dynamic objects
  • New gravity blocks
  • The "Add Velocity To Players" option for rotating blocks (e.g. poker roll) now supports any rotation
  • New buttons to lock movement on selected axis while moving/cloning blocks
how to create levels
1. Read the Documentation

Reading the editor documentation will save you a lot of time when creating your levels.

You will learn:

  • Navigation
  • Level Structure
  • Player physics
  • Editor Interface

and much more...

📗read documentation
2. Explore the Editor

Start by playing around with the editor and familiarising yourself with its interface and functions.

Start small and test often, you can always save your work in progress to continue later or to share it with your friends.

Ask questions, exchange tips and gather feedback in the LOLBeans Discord.

📐start creating
3. Publish Levels

Ready for prime time? Submit your level for publishing.

Following these guidelines improves your chances of being published:

  • Create fair challenges
  • Balance difficulty
  • Make levels enjoyable
  • Help players focus
